# 🧑‍💻 Project Tracker Dashboard

A modern project management dashboard built with **React + TypeScript + Tailwind CSS**.  
It supports Kanban, List (sortable), and Timeline (Gantt-style) views.

---

🔗 Live Demo

👉 https://anilvelozityglobal.vercel.app

---

## 🚀 Features

- ✅ Kanban board with drag-and-drop
- ✅ List view with sorting (Title, Priority, Due Date)
- ✅ Inline status update via dropdown
- ✅ Timeline / Gantt view with priority-based color bars
- ✅ Virtualized list rendering for performance
- ✅ Zustand-based state management

---

🛠️ Tech Stack
⚛️ React vite
🟦 TypeScript
🎨 Tailwind CSS
🐻 Zustand (State Management)
⚡ Vite
